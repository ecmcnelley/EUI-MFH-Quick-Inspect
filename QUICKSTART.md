## âš ï¸ Troubleshooting

### Page Won't Load
1. Wait 2 minutes after enabling GitHub Pages
2. Hard refresh: **Ctrl+Shift+R** (Win) or **Cmd+Shift+R** (Mac)
3. Try incognito/private mode

### Dropdowns Empty
1. Make sure you're using latest `index.html`
2. Clear browser cache
3. Check browser console (F12) for errors

### Can't Take Photos
1. Allow camera permissions
2. Use HTTPS (GitHub Pages provides this)
3. Try file upload instead

### Storage Full
1. Generate and save current report
2. Click **ğŸ”„ Start New Inspection**
