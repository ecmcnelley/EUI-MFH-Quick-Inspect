# Rent Inspection System

Web-based property inspection system for residential units.

## Quick Start

Open `index.html` in your browser to start using the inspection system.

## Features

- Property database with 16 pre-configured properties
- Auto-population of addresses, inspectors, and contact information
- Dynamic unit layout configuration
- Safety inspection tracking
- Appliance inspection with specialized fields
- Room-by-room detailed inspection
- Photo documentation
- Professional report generation
- Local storage for data persistence

## Deployment

This is a single-file application. No server or build process required.

For GitHub Pages:
1. Enable Pages in Settings
2. Select main branch
3. Access via: `https://[username].github.io/[repo-name]/`

## Browser Compatibility

Works in all modern browsers (Chrome, Firefox, Safari, Edge).
