# Rent Inspection System

A comprehensive web-based property inspection system for residential units.

## Features

- **Property Database** - 16 pre-configured properties with auto-population
- **Dynamic Layout Configuration** - Configure bedrooms, bathrooms, and room types
- **Safety Inspection** - Smoke/CO alarm tracking and documentation
- **Appliance Tracking** - Complete appliance inspection with special Water Heater fields
- **Room-by-Room Inspection** - Detailed forms for each room type
- **Photo Documentation** - Upload and attach photos to inspection items
- **Professional Reports** - Generate printable PDF reports
- **Local Storage** - Auto-save functionality to resume inspections
- **Customization** - Upload logo, customize colors and branding

## Usage

1. Open the application in your browser
2. Customize branding (logo, colors, text)
3. Enter property information or select from database
4. Configure unit layout
5. Document safety items and appliances
6. Inspect each room
7. Generate and print report

## Browser Compatibility

- Chrome/Edge (recommended)
- Firefox
- Safari
- Mobile browsers supported

## Technical Details

- Single-file application (no dependencies)
- All data stored in browser localStorage
- No server required
- 100% client-side JavaScript
