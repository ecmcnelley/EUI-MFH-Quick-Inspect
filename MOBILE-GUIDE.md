## üêõ Troubleshooting

### Camera Won't Open

**Try:**
1. Check permissions in Settings
2. Grant camera permission
3. Restart app
4. Try different browser
5. Use file upload instead

### Photos Not Showing

**Try:**
1. Check storage space
2. Reduce photo resolution
3. Take fewer photos
4. Clear browser cache
5. Restart device

### App Feels Slow

**Try:**
1. Close other apps
2. Restart device
3. Clear browser data
4. Reduce photo count
5. Update browser/OS

### Data Not Saving

**Try:**
1. Check storage space
2. Allow storage permission
3. Don't use private browsing
4. Manual save frequently
5. Export reports immediately

### Keyboard Issues

**Try:**
1. Update keyboard app
2. Try different keyboard
3. Restart device
4. Clear keyboard cache
5. Disable autocorrect if buggy

### Screen Orientation

**Try:**
1. Enable auto-rotate
2. Check orientation lock
3. Rotate device fully
4. Restart app
5. Update OS if old
